# Solar System Simulation

A small Javascript web app containing a recreation of the solar system in ThreeJS, including the sun along with every planet. 
Each planet is designed to revolve around the sun, along with having their own moons constantly orbiting them. 
Additionally, the user can pause this rotation and control it on their own by dragging around the Earth.
